# Spring Cloud Matrix

## 介绍

Spring cloud matrix是微服务的矩阵，整合目前主流的微服务框架

### 项目环境

开发环境和相应中间件版本

- jdk11
- gradle7.4.2
- mysql8.0
- redis5.0
- elasticSearch8.0
